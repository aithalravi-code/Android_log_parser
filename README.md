# Android Log Viewer

A modern, minimal HTML/JS tool for analyzing Android debug logs locally.

## Features
- Upload a ZIP of Android debug logs
- Extract and combine all log/text files
- Multi-service and time-range filtering (toggle icons)
- Live log view with color highlights per service
- Export filtered logs
- BLE log viewer tab with decoding
- Modern, minimalist UI

## Usage
1. Open `index.html` in your browser
2. Upload your Android debug log ZIP
3. Use filters and export as needed

No server or build tools required.
