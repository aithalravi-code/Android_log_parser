# Production

This directory contains production-ready, stable source code.

## Contents

- `src/` - Production source code
- `index.html` - Application entry point

## Guidelines

- All code here must be thoroughly tested
- Code must pass all unit and regression tests
- Code should be optimized and documented
- Ready for deployment

## Development Workflow

1. Develop features in `Development/`
2. Test thoroughly
3. Move to `Production/` when stable
4. Deploy from `Production/`
